Visual Studio Code is a lightweight but
powerful source code editor which runs on
your desktop and is available for Windows,
OS X and Linux. It comes with built-in support
for JavaScript, TypeScript and Node.js and has
a rich ecosystem of extensions for other languages
(C++, C#, Python, PHP) and runtimes.
